# Marketing / CTA Page

**Status:** Draft
**Version:** v0.1.0
**Owner:** Aphrodite

## What this is

A reusable conversion-focused marketing archetype for pages with one primary ask, where the user already has enough context that the page can stay narrower and more direct than a full landing page.

## When to choose this

- The page has one clear conversion target.
- The user is already partly informed and needs a sharper push to act.
- The page benefits from limited scope and minimal navigation.

## When not to choose this

- The page needs broad brand introduction or extensive storytelling.
- The user needs pricing comparison depth.
- The page is really a homepage, blog article, or content hub.

## Required sections

1. Focused value statement with CTA above the fold.
2. Supporting proof or visual context.
3. Benefits or objection-handling section.
4. Trust reinforcement near the conversion point.

## Allowed variations

- Copy-led or visual-led top section.
- Trust treatment via testimonials, logos, or metrics.
- Optional FAQ or risk-reduction block.
- Optional lower-commitment secondary CTA.

## Important states

- Default loaded state.
- Video or demo-playing state if media is present.
- CTA hover and active states.
- Mobile stacked state.

## UX rationale

The narrower the objective, the less room there is for distraction. This pattern keeps the promise, proof, and action path close together so users do not have to reorient repeatedly before acting.

## Implementation notes

- Keep the primary CTA visible early and repeat only when it clarifies the flow.
- Use proof near the conversion point, not buried below generic filler.
- Avoid extra navigation that competes with the action path.
- Mobile should keep the action comfortably reachable without losing proof context.
